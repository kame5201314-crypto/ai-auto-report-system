## 三、SaaS 化路徑 (Phase 2)

### 方案 A：抽成制

```
┌─────────────────────────────────────────┐
│           交易抽成定價                    │
├─────────────────────────────────────────┤
│  月交易額           技術服務費率           │
├─────────────────────────────────────────┤
│  < NT$ 100,000      1.0%                │
│  NT$ 100K ~ 500K    0.8%                │
│  NT$ 500K ~ 1M      0.6%                │
│  > NT$ 1,000,000    0.5%                │
└─────────────────────────────────────────┘

* 不含藍新金流手續費 (約 2.0% ~ 2.8%)
```

**優點：**
- 低門檻，按使用量付費
- 商家無風險試用
- 與商家利益一致

**缺點：**
- 高交易量商家成本上升
- 需要精確的交易追蹤

### 方案 B：訂閱制

```
┌─────────────────────────────────────────┐
│           月訂閱方案                      │
├──────────┬──────────┬───────────────────┤
│  方案    │  月費    │  包含內容          │
├──────────┼──────────┼───────────────────┤
│  Starter │  NT$ 0   │  100 筆/月        │
│  Pro     │  NT$ 990 │  1,000 筆/月      │
│  Business│ NT$ 2,990│  10,000 筆/月     │
│  Enterprise│ 洽談   │  無限制           │
└──────────┴──────────┴───────────────────┘
```

**優點：**
- 收入可預測
- 高交易量商家成本固定
- 簡化計費邏輯

**缺點：**
- 入門門檻較高
- 低交易量商家可能流失

### 建議策略：混合模式

```
┌─────────────────────────────────────────┐
│           混合定價策略                    │
├─────────────────────────────────────────┤
│                                         │
│  1. 免費層 (Starter)                     │
│     - 100 筆/月免費                      │
│     - 超過部分 1.0% 抽成                 │
│                                         │
│  2. 訂閱層 (Pro/Business)                │
│     - 固定月費包筆數                      │
│     - 超過部分 0.5% 抽成                 │
│                                         │
│  3. 企業層 (Enterprise)                  │
│     - 客製報價                           │
│     - SLA 保證                          │
│     - 專屬技術支援                        │
│                                         │
└─────────────────────────────────────────┘
```
